# CSPanel
This extension is a recreation of the real Roblox Internal Customer Service Panel. It also has its own database.
Source Code is literally open source.

Setup:
  1. Press on Code and then "Download as ZIP".
  2. Extract the ZIP
  3. Go to chrome://extensions/ on your browser
  4. Press "Load unpacked" and select your extracted folder
  5. Go to roblox.com and you will see a button in the right corner (down), and click it to open the CS Panel.

This was made by VAK.
Enjoy.
